navbar-expand-lg: Cette classe spécifie le comportement de la barre de navigation en fonction de la taille de l'écran. Dans ce cas, "lg" signifie "large", ce qui indique que la barre de navigation s'étendra sur toute la largeur de l'écran pour les écrans de taille large et supérieure

navbar-dark: Cette classe définit le thème sombre pour la barre de navigation.

bg-dark: Cette classe définit la couleur de fond de la barre de navigation comme étant sombre. 

fixed: Cette classe définit le positionnement de la barre de navigation. "fixed" signifie que la barre de navigation restera fixe en haut de la fenêtre du navigateur même lorsque l'utilisateur fait défiler la page

collapse: Cette classe est utilisée pour indiquer que le contenu de la barre de navigation doit être (réduit) 

justify-content-end:  est utilisée pour aligner les éléments de la barre de navigation à l'extrémité droite de la barre de navigation.
